Twitter Bot

Details:
  -This twitter bot will search through a specific hashtag on twitter and do the following:
    -Follow the user
    -Favorite the tweet
    -Retweet the tweet
  -The twitter bot has a feature that will check to see if you have already retweeted a specific tweet, 
  then it skips the tweet to save time.
